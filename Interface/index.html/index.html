Vamos adicionar imagens

É necessário identificar os direitos altorias da imagens. Dessa maneira vamos listar alguns sites para realizar a busca de imagens: Pexels, unsplash e pixabay

Ao buscar imagens no google é interessante atentar-se aos tamanhos de imagens, a sua cor podendo ser transparente, branco e etc.

Para imagens com mais detalhes, melhor qualidade e tranparência, use PNG

Para imagens com mmenos qualidade, mas com melhor compactação JPEG

Podemos usar o Gimp para fazer o tratame nto das imagens
